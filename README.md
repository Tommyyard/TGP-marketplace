# Marketplace Website Project 4

Project ini adalah website template marketplace dengan fitur kategorisasi template berdasarkan jenisnya.

## Fitur

- **Sidebar Kategori**:Navigasi untuk memilih kategori template
- **Show/Hide Content**: Klik kategori di sidebar untuk menampilkan template sesuai kategori
- **5 Kategori Template**:
  1. E-Commerce - Template toko online
  2. Personal & Portofolio - Template portofolio personal
  3. Company Profile - Template profil perusahaan
  4. Blog & Media - Template blog dan media
  5. Landing Page - Template halaman landing

## Struktur File

```
├── index.html    # File HTML utama
├── style.css     # File styling (CSS)
├── main.js       # File JavaScript (logika show/hide)
└── README.md     # Dokumentasi project
```

## Cara Menggunakan

1. Buka `index.html` di browser
2. Klik kategori di sidebar untuk mengganti tampilan template
3. Template E-Commerce ditampilkan secara default saat pertama kali dibuka

## Teknologi

- HTML5
- CSS3 (Flexbox)
- JavaScript (Vanilla)

## Catatan Pengembangan

- Menggunakan selector yang sudah ada di HTML (`.side-bar p`, `.btn-desain`)
- Kode JavaScript dibuat manual (satu per satu) untuk memudahkan pemahaman pemula
- Setiap baris kode memiliki komentar dalam Bahasa Indonesia
